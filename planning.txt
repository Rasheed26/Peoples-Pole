    Routes means end points
      1> user authentication
        --> /signup : POST -- create a new user account
        --> /login : POST -- login to am existing account
      2> voting
        --> /candidate : GET -- Get the list of candidates
        --> /vote/candidateId : POST -- vote for a specific candidate

      3> Vote counts:
        --> /profile: GET -- get the users profile information
        --> /profile/password: PUT -- change the users password

      4> Admin Candidate Managemnet:
        --> /candidates : POST -- Create new  candidate
        --> /candidates/:candidated : PUT -- Update an existing candidate
        --> /candidates/:candidated : DELETE -- Delete a candidate from the last
